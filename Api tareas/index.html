Acerca de TaskManager

TaskManager es una aplicación web completa para la gestión de tareas. Utiliza una API RESTful para operaciones CRUD (Crear, Leer, Actualizar, Eliminar) de tareas.

Características:

  • Agregar nuevas tareas con nombre y descripción
  • Actualizar estado de tareas (pendiente/completada)
  • Editar tareas existentes
  • Eliminar tareas
  • Filtrar por estado
  • Interfaz responsive y moderna
